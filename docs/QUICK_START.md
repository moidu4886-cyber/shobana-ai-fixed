# Quick Start Guide for Advanced AI Search

## Introduction
This guide will help you quickly get started with the advanced AI search feature. 

## How to Use Advanced AI Search
1. **Send a Query**: You can send a query in natural language (e.g., "Tovino superhero movie").
2. **Spell Correction**: The bot will automatically correct any spelling mistakes in your query.
3. **Searching Database**: After correcting the spelling, the bot will search the database for relevant results.
4. **Displaying Results**: The bot will display the results in an easy-to-understand format.
5. **Processed File**: Finally, the bot will send you a processed file that contains cleaned metadata related to your query.

## Example Queries
- `Tovino superhero movie`
- `Theri 2026 Tamil`

## Example Workflow
1. User sends the query: **"Tovino superhero movie"**.
2. Bot corrects the query (if necessary) and searches the database.
3. Bot finds relevant data and shows results:
   - Movie Title: Tovino's Next Adventure
   - Genre: Superhero
   - Release Year: 2026
4. Bot sends a processed file with cleaned metadata:
   - Title: Tovino's Next Adventure
   - Genre: Superhero
   - Year: 2026

This demonstrates how the advanced AI search facilitates quick and accurate retrieval of information.
